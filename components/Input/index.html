Input

The Input component is a component that is used to get user input in a text field.

Usages

This is basic Input Component from our component library.

                
                  < -- For Default Input Component -->
                  

                  <div class="input-box"> 

                  <label>Enter Name </label> 

                  <input placeholder="Please Enter Text" type="text" class="input" />

                  </div>
