Alert

Alerts are used to communicate a state that affects a system, feature or page.

Usages

This is Primary alert message from Componently.
This is Success alert message from Componently.
This is Warning alert message from Componently.

To achieve the above alert, you need to follow the following code: 

                
                  < -- For Primary Alert -->
                  

                  <div class="alert">Alert</div>
                  

                  

                  < -- For Success Alert -->
                  

                  <div class="alert alert-success">Success Alert</div>
                    

                  

                  < -- For Warning Alert -->
                  

                  <div class="alert alert-warning">Warning Alert</div>
